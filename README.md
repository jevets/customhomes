## chofasheville

Static site generated with @11ty/eleventy

### Commands

`yarn dev`

Watches CSS & HTML files and recompiles; serves locally on localhost:8080 (default port).

`yarn build`

Compiles CSS & HTML into `./dist`

`yarn serve`

Watches & serves only the HTML; useful when you're not working on CSS - quicker than watching both CSS & HTML.

`yarn css`

Compiles only the CSS (tailwindcss) to `./dist/css/styles.css`
